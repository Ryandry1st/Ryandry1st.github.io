Getting Started with Python

In this tutorial we will go through the basics of getting started with Python. We will start by going through the installation and setup of Anaconda. Then we will learn about the basics of jupyter notebooks and write your first Python code. We will continue with some basics for working with data and extracting information using pandas and numpy. At the end of the tutorial you will have a working installation and basic understanding of using python for tabular data.
